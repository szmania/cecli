- [ ] **Command to Run:** `venv/scripts/python -m aider --message "/tools"`

- [ ] **Action:** If it fails or shows incorrect output, I will investigate `aider/commands.py` (the `cmd_tools` function) and `aider/coders/agent_coder.py` to fix the issue.

## 2. Test `/tools-create` Command
- [ ] **Command to Run:** `venv/scripts/python -m aider --message "/tools-create my_test_tool.py 'a simple test tool that prints hello'"`
- [ ] **Expected:** A new tool file `my_test_tool.py` should be created in the `.aider/tools/` directory, and the output should confirm its creation and loading.
- [ ] **Action:** If creation or loading fails, I will debug `aider/commands.py` (`cmd_tools_create`) and the `aider/tools/create_tool.py` script.

## 3. Test `/tools-load` Command
- [ ] **Command to Run:** `venv/scripts/python -m aider --message "/tools-load .aider/tools/my_test_tool.py"`
- [ ] **Expected:** The output should confirm the tool has been loaded successfully. We will verify this by running `/tools` again.
- [ ] **Action:** If loading fails, I will investigate `aider/commands.py` (`cmd_tools_load`) and `aider/tool_manager.py`.

## 4. Test `/tools-unload` Command
- [ ] **Command to Run:** `venv/scripts/python -m aider --message "/tools-unload MyTestTool"`
- [ ] **Expected:** The output should confirm the tool has been unloaded.
- [ ] **Next Command:** `venv/scripts/python -m aider --message "/tools-unload .aider/tools/my_test_tool.py"` (to test unloading by path).
- [ ] **Action:** If unloading fails, I will debug `aider/commands.py` (`cmd_tools_unload`) and `aider/tool_manager.py`.

## 5. Test `/tools-move` Command
- [ ] **Command to Run:** `venv/scripts/python -m aider --message "/tools-move MyTestTool global"`
- [ ] **Expected:** The tool's file should be moved from the local `.aider/tools` directory to the global `~/.aider/tools` directory.
- [ ] **Next Command:** `venv/scripts/python -m aider --message "/tools-move MyTestTool local"` (to move it back).
- [ ] **Action:** If moving fails, I will investigate `aider/commands.py` (`cmd_tools_move`) and `aider/tool_manager.py`.

## 6. Test `/tools-edit` Command

- [ ] **Expected:** The `my_test_tool.py` file should be added to the chat as an editable file, and the session should continue, allowing for edits.
- [ ] **Action:** If the file is not added correctly, I will debug `aider/commands.py` (`cmd_tools_edit`).
